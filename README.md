# HSK Stacku Gappu

A from-scratch rebuild of the namesake mod for the RimWorld HSK project.

## Features

- Stack limit with non-linear scale
- Separate limits for items and categories
- Similar stack count limit per cell
- Multistack: disabled, globally, or per storage
- Optional forced ejection when limits are lowered
- Limit by item count or by percentage
- Custom highlight color for modified limits

---

Переписанный с нуля ребилд одноименного мода для проекта RimWorld HSK.

## Особенности

- Лимит стопок с нелинейной шкалой
- Отдельные лимиты для предметов и категорий
- Лимит количества похожих стопок в клетке
- Мультистак: выключен, глобально или для каждого склада
- Опциональное принудительное извлечение при уменьшении лимитов
- Ограничение по количеству предмета или в процентах
- Настраиваемый цвет подсветки изменённых лимитов
